CSS a RWD

  • CSS - cascading style sheets (kaskadowe arkusze stylów), to język dzięki któremu zmieniamy część wizualną naszych stron WWW
  • RWD - responsive web design (responsywne projektowanie stron internetowych) - technika projektowania stron WWW w taki sposób, aby automatycznie dostosowywały się do okna przeglądarki np. smartfonów, czy tabletów.
